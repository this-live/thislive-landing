Quickstart

Run Maestro locally

Free, local-first, bring your own keys. Clone the public repo and bring it up with Docker.

git clone https://github.com/brycemurad0/maestro
cd maestro
docker compose up -d  # Vega routing + Quorum, on your machine
100% private when local. Run it fully local and your data never leaves your hardware. Use BYOK to reach frontier models — the secret-scan redaction runs before anything leaves.
